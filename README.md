# Identity-Based-Network-Security-System-BNS-ZTA-
This project is a web-based application that demonstrates a secure user authentication system. It includes core features such as user registration, login functionality, email-based One-Time Password (OTP) verification, and CAPTCHA protection to prevent automated access.
